# HTML Advanced

![Background](template/background.png)

## Description

This project covers the fundamentals of HTML by building the structure of a web page from scratch using semantic HTML5 elements. No CSS or JavaScript is applied — the focus is purely on the HTML skeleton and proper document structure.

## Project Structure

```
html_advanced/
├── template/          # Design assets (images, mockups)
├── task_1/            # Basic HTML page structure
├── task_2/            # Header and footer sections
├── task_3/            # Main content sections
├── task_4/            # Navigation and links
├── task_5/            # Images and media
├── task_6/            # Tables and lists
└── task_7/            # Complete page structure
```

## Learning Objectives

- Understand what HTML is and how to create a valid HTML page
- Use semantic HTML5 elements (`header`, `main`, `footer`, `section`, `article`, etc.)
- Structure a web page using headings, paragraphs, links, images, and lists
- Follow W3C standards for HTML markup

## Requirements

- All files must be W3C compliant and validate with the [W3C Validator](https://validator.w3.org/)
- No CSS or JavaScript allowed in this project
- All HTML files must end with a newline
